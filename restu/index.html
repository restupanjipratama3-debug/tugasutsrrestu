Informasi Dasar

Web ini adalah dashboard sederhana berbasis HTML, CSS, dan JavaScript. Dashboard ini menampilkan konten interaktif yang dapat berganti sesuai tombol yang diklik di navbar atas tanpa memuat ulang halaman (single-page interface sederhana). Web ini dibuat untuk menampilkan data atau informasi dasar secara interaktif dan mudah diakses, cocok untuk belajar membuat dashboard sederhana atau demo portofolio.
